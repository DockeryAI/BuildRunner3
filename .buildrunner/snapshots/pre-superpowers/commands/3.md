# /3000 - Universal Dev Server Starter

Kill anything on port 3000 and start the current project's dev server.

## Steps

1. Kill any process on port 3000:
```bash
lsof -ti:3000 | xargs kill -9 2>/dev/null || true
```

2. Detect project type and start appropriate dev server:

**Detection order:**
- If `package.json` exists → check for dev script → `npm run dev`
- If `Cargo.toml` exists → `cargo run`
- If `go.mod` exists → `go run .`
- If `requirements.txt` or `pyproject.toml` exists → check for common Python frameworks
- If `Makefile` exists with `dev` target → `make dev`

3. Run the dev server in the foreground so user can see output.

4. If no recognizable project type, report what was found and ask user what command to run.

**Default for most web projects:** `npm run dev`
